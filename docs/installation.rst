============
Installation
============


Stable release
--------------

To install python-selenium-php-travels, run this command in your terminal:

.. code-block:: console

    $ pip install python_selenium_php_travels

This is the preferred method to install python-selenium-php-travels, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for python-selenium-php-travels can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/trinhphat001/python_selenium_php_travels

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/trinhphat001/python_selenium_php_travels/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/trinhphat001/python_selenium_php_travels
.. _tarball: https://github.com/trinhphat001/python_selenium_php_travels/tarball/master
